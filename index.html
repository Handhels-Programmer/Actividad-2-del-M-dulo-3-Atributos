Identifica y describe las propiedades de los siguientes elementos CSS:

Tema 1 - Texto

Propiedad Valor Inicial Descripción
Color Depends on user agent (generalmente black). Define el color del texto dentro de un elemento. Puedes especificar el valor en distintos formatos como nombre de color (e.g. Red), valores hexadecimales (e.g. #ff0000), RGB (e.g. Rgb(255, 0, 0)), y otros.
Font-family serif (esto puede variar según el navegador). Especifica la fuente que se usará para el texto dentro de un elemento. Puedes definir una fuente específica o una lista de fuentes de respaldo para garantizar que se muestre correctamente en diferentes sistemas.
Font-size medium (generalmente 16px en la mayoría de los navegadores). Define el tamaño de la fuente para el texto dentro de un elemento. Se puede especificar en diferentes unidades, incluyendo píxeles (px), ems (em), rems (rem), porcentajes (%), entre otros.
Font-weight normal (equivale a 400). Especifica el grosor del texto. Los valores pueden ser palabras clave como normal, bold, bolder, lighter, o valores numéricos que van desde 100 hasta 900, donde 400 representa el peso normal y 700 representa el peso en negrita.
Font-style normal. Especifica el estilo de la fuente, permitiendo que el texto se muestre en diferentes estilos como normal, cursiva o negrita.
Font-variant normal. Controla la variación de la fuente, específicamente si se debe utilizar una versión "pequeña mayúscula" (small-caps) de la fuente.
Font inherit. La propiedad font es una propiedad abreviada que permite establecer múltiples propiedades de tipografía en una sola declaración.
Text-align start (izquierda o derecha dependiendo del idioma). Controla la alineación del texto dentro de un elemento de bloque.
Line-height normal. Establece la altura de la línea para el texto en un elemento.
Text-decoration none. Se utiliza para especificar el estilo de decoración del texto, como subrayados o tachados.
Text-transform none. Controla la capitalización del texto en un elemento.
Vertical-align baseline. Controla la alineación vertical de un elemento en relación con su línea base.
Text-indent 0. Define la cantidad de espacio que se debe dejar al inicio de la primera línea de un bloque de texto.
Letter-spacing normal. Controla el espacio adicional entre los caracteres de un texto.
Word-spacing normal. Controla el espacio adicional entre las palabras en un bloque de texto.
White-space normal. Controla cómo se maneja el espacio en blanco dentro de un elemento.

Tema 2 - Enlaces

Característica Propiedades Descripción
Tamaño, Color y Decoración font-size Define el tamaño de la fuente de un elemento.
color Especifica el color del texto dentro de un elemento.
text-decoration Controla la decoración del texto, permitiendo subrayados, tachados, etc.
line-height Establece la altura de la línea, afectando el espaciado vertical entre líneas de texto.
letter-spacing Ajusta el espacio entre los caracteres de un texto.
word-spacing Ajusta el espacio entre las palabras en un bloque de texto.
text-align Controla la alineación del texto dentro de un contenedor (izquierda, derecha, centro, justificar).
Decoración Personalizada text-shadow Aplica una sombra al texto, permitiendo efectos visuales personalizados.
text-transform Controla la capitalización del texto (mayúsculas, minúsculas, etc.).
font-family Define la familia de fuentes a utilizar para el texto.
background-color Establece el color de fondo detrás del texto, afectando la decoración general.
text-indent Define el espacio que se deja al inicio de la primera línea de un bloque de texto, creando una sangría.
vertical-align Controla la alineación vertical de elementos en línea con respecto a su línea base.
opacity Controla la opacidad de un elemento, permitiendo efectos de transparencia.

Tema 3 - Imágenes

Característica Propiedades Descripción
Anchura y Altura width Define el ancho de un elemento. Puede ser especificado en unidades absolutas (px, cm) o relativas (%).
height Define la altura de un elemento. Similar a width, puede ser en unidades absolutas o relativas.
max-width Establece el ancho máximo que un elemento puede alcanzar, útil para hacer que un diseño sea responsivo.
max-height Establece la altura máxima de un elemento.
min-width Define el ancho mínimo de un elemento, evitando que se reduzca a menos de un cierto tamaño.
min-height Define la altura mínima de un elemento.
box-sizing Controla cómo se calculan las dimensiones de un elemento, ya sea incluyendo o excluyendo el padding y los bordes.
Eliminar el Borde border Establece un borde alrededor de un elemento, incluyendo propiedades como border-width, border-style y border-color.
border-width Define el grosor del borde de un elemento (ej. 1px, 2px).
border-style Especifica el estilo del borde (ej. solid, dashed, none).
border-color Define el color del borde de un elemento.
outline Especifica un contorno alrededor de un elemento, que no ocupa espacio en el flujo del documento.
outline-width Define el grosor del contorno.
outline-style Especifica el estilo del contorno.
outline-color Define el color del contorno.

Tema 4 - Listas

Propiedad Valor Inicial Descripción
list-style-type disc Especifica el tipo de marcador de la lista.
list-style-position outside Define la posición de los marcadores en una lista.
list-style-image none Permite establecer una imagen como marcador de lista.
list-style none Es una propiedad abreviada que combina list-style-type, list-style-position, y list-style-image.

Tema 5 - Tablas

Propiedad Se Aplica a Valor Inicial Descripción
border-collapse table, th, td separate Controla el colapso de los bordes de las celdas de una tabla.
border-spacing table 0 Define el espacio entre los bordes de las celdas de una tabla.
empty-cells table, th, td show Controla la visualización de celdas vacías en una tabla.
caption-side table top Especifica la posición de la leyenda de la tabla.
